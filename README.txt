======= Небольшой инструмент для упрощения процесса собеседования =======
=========================================================================
Для начала установите Python 3.9.6 или выше и все библиотеки из файла “requirements.txt” ======================================================
=========================================================================
На главной странице вы может прочесть информацию об авторах, о приложении; сменить тему и начать собеседование =========================
=========================================================================
После нажатия на кнопку “Начать собеседование вы должны пройти регистрацию, вписав в поле “Название компании” полное наименование организации, в которой проходить собеседование, в поле “ФИО кандидата” полные Фамилию, Имя и Отчество кандидата, в поле “Дата рождения” дату рождения кандидата ======================================================
=========================================================================
Затем вам будет представлено несколько вопросов с критериями оценивания. Вы должны прочитать вопрос кандидату, выслушать ответ и оценить его по 10-ти балльной шкале, исходя из критериев оценивания ==================== =========================================================================
После прохождения всех вопросов программа выдаст вам процент пригодности кандидата к работе ======================================================
=========================================================================

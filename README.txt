Job Interview
Небольшой инструмент для упрощения жизни HR специалистов

Для начала установите Python 3.9.6 или выше и все библиотеки из файла “requirements.txt” 

На главной странице вы может прочесть информацию об авторах; импортировать/экспортировать макеты; сменить тему; 

После нажатия на кнопку “Выбрать макет", открывается окно в котором вы можете выбрать один из макетов; удалить выбранный макет; 
изменить выбранный макет; начать собеседование с выбранным макетом; создать новый макет.

После нажатие кнопки "Изменить макет" появится диалоговое окно, в котором можно будет обучить макет/изменить вопросы макета.

После нажатия на кнопку “Начать собеседование", кандидату будет предложено тестирование, по результатам которого, программа выдаст время, которое кандидат у вас проработает.

Важно! Программа дает лишь примерную оценку. Основное решение лежит на HR-специалисте
